# Elegant Yacht Tours Website

A multi-page responsive website concept for a luxury yacht tours brand, built with HTML and CSS.

## Pages
- Home: `index.html`
- Tours: `tours.html`
- Gallery: `gallery.html`
- About: `about.html`
- Contact: `contact-us.html`

## Tech Used
- HTML5
- CSS3
- Google Fonts (Montserrat)
- Font Awesome icons

## Project Structure
├── index.html
├── tours.html
├── gallery.html
├── about.html
├── contact-us.html
├── style.css
└── images/
     └── yacht.jpg

## How to Run
Open `index.html` in any modern web browser.
